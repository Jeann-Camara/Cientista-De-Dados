Descrição¶

Para facilitar a administrao da segurana pblica, o Departamento de Polcia de Seattle dividiu a cidade em 5 partes, cada uma com uma delegacia. Cada delegacia foi subdividida em setores, e estes foram divididos em beats (hondas). A administrao tem dois datasets um chamado Call_Data, para obter maiores informaes acesse este link e outro chamado Crime_Data, maiores detalhes aqui (acesse os links para compreender o dicionrio de cada dataset).

O download do dataset de chamadas est disponvel neste link e o download do dataset de crimes est aqui.

O administrador geral pediu que voc, cientista de dados, o ajudasse a elaborar um report respondendo algumas perguntas. O objetivo deste projeto ajudar a administrao fornecendo informaes relevantes para ajud-los a resolver de forma mais rpida uma chamada, um crime ou mesmo replanejar a distribuio das hondas e setores. Como bom cientista de dados que voc , voc tem a liberdade de ir alm e responder mais perguntas do que as que esto aqui =D, atravs dos insights que vo surgindo a medida que voc mergulha no problema.

from datetime import datetime

def num_dia(dataVar):
    datetime_object = datetime.strptime(dataVar, '%m/%d/%Y %I:%M:%S %p')
    return datetime_object.day

def num_semana(dataVar):
    datetime_object = datetime.strptime(dataVar, '%m/%d/%Y %I:%M:%S %p')
    return str(datetime_object.isocalendar()[1]) + '_' + str(datetime_object.isocalendar()[0])

def num_ano(dataVar):
    datetime_object = datetime.strptime(dataVar, '%m/%d/%Y %I:%M:%S %p')
    return datetime_object.year

def num_mes(dataVar):
    datetime_object = datetime.strptime(dataVar, '%m/%d/%Y %I:%M:%S %p')
    return datetime_object.month

def dia_da_semana(dataVar):
    datetime_object = datetime.strptime(dataVar, '%m/%d/%Y %I:%M:%S %p')
    if datetime_object.isoweekday() == 1:
        return 'Monday'
    elif datetime_object.isoweekday() == 2:
        return 'Tuesday'
    elif datetime_object.isoweekday() == 3:
        return 'Wednesday'
    elif datetime_object.isoweekday() == 4:
        return 'Thursday'
    elif datetime_object.isoweekday() == 5:
        return 'Friday'
    elif datetime_object.isoweekday() == 6:
        return 'Saturday'
    elif datetime_object.isoweekday() == 7:
        return 'Sunday'
    else:
        return 'ERROR'

def periodo_dia(dataVar):
    datetime_object = datetime.strptime(dataVar, '%m/%d/%Y %I:%M:%S %p')
    if (datetime_object.hour < 6):
        return 'After Midnight'
    elif (datetime_object.hour < 12):
        return 'Morning'
    elif (datetime_object.hour < 18):
        return 'Afternoon'
    else:
        return 'Night'

def time_diff(inicio, fim):
    datetime_inicio = datetime.strptime(inicio, '%m/%d/%Y %I:%M:%S %p')
    datetime_fim = datetime.strptime(fim, '%b %d %Y %I:%M:%S:%f%p')
    
    if datetime_fim >= datetime_inicio:
        diferenca = datetime_fim - datetime_inicio
    else:
        diferenca = -1
    return diferenca
